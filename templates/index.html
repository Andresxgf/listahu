¿Qué es?

Este proyecto nace para crear una base de datos de mensajes y llamadas no deseadas, sean SPAM, extorsión o estafa, para identificar y eventualmente para poder bloquearlos, una lista negra.

La idea

Como cada día hay noticias de estafas, extorsiones y otros crímenes realizados a través de mensajes o llamadas a celulares y la gente sigue cayendo Lista Hũ apunta a identificar las lineas de que están haciendo esto. Recabar estos datos solo se puede lograr con el esfuerzo colectivo. Actualmente se puede descargar la lista y consultar los números pero la idea es un futuro cercano hacer una app que se actualice con los datos de la lista para bloquear automáticamente los mensajes y llamadas molestas o potencialmente peligrosas.

Para agregar un número basta con llenar el formulario con el número y una captura de pantalla del mensaje o si fue una llamada al menos la captura de pantalla del registro de llamadas.

A futuro

Además de la app para bloquear automáticamente la idea es poder descargar la lista filtrado por tipo de denuncia, también ofrecer un sistema de votaciones para calificar como válida o no la denuncia, descargar solo las denuncias verificadas.
